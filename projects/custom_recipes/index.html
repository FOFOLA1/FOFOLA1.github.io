Art of Code [ALPHA 0.1.0] - Rainmeter skin

Rainmeter is a free, open-source tool for creating visualizations and dashboards.

I created a Rainmeter skin for displaying component usage or time in nice looking coding style. Now it can display current time, search input powered by Brave, CPU and RAM usage, and battery percentage.

It's still in development and I plan to add color settings, change search engine, another widgets like music control, disk usage.

For design I was inspired by wallpaper created by Probler.
