Art of Code [ALPHA 0.2.0] - Rainmeter skin

Rainmeter is a free, open-source tool for creating visualizations and dashboards.

I created a Rainmeter skin for displaying component usage, time and more in nice looking coding style.

It's still in development and I plan to add color settings, option to change search engine, another widgets like music control, disk usage.

For design I was inspired by wallpaper created by Probler.

Widgets currently available

  • Clock - displays time with date
  • System usage - shows CPU, GPU and RAM usage with battery percentage
  • Music player - displays info about currently playing song, you can also pause/play, song and go to the previous or next song
  • Search bar - allows you to search through the internet from your desktop, powered by Brave search engine
